# Portfólio Pessoal - Landing Page

Este projeto é uma landing page pessoal desenvolvida como atividade acadêmica. O objetivo do projeto é aplicar na prática os conceitos de desenvolvimento web Front-end, criando um portfólio interativo, responsivo e com funcionalidades dinâmicas.

## Funcionalidades

O site conta com as seguintes seções e interações:

* **Sobre Mim:** Apresentação pessoal, tecnologias de estudo e hobbies.
* **Formação:** Histórico acadêmico e cursos complementares.
* **Portfólio:** Galeria de projetos desenvolvidos com links diretos para os repositórios.
* **Contato:** Formulário interativo com validação de campos vazios e de formato de e-mail usando Expressões Regulares (Regex).
* **Tema Claro/Escuro (Dark Mode):** Alternância de paleta de cores da página via JavaScript, melhorando a acessibilidade e a experiência do usuário.
* **Menu Responsivo (Mobile):** Navegação adaptada para dispositivos móveis com menu "hambúrguer" dinâmico.

## Tecnologias Utilizadas

Este projeto foi construído puramente com as tecnologias base da web (Vanilla):

* **HTML5:** Estruturação semântica.
* **CSS3:** Estilização, layout Flexbox, variáveis nativas para o tema dinâmico e Media Queries para responsividade.
* **JavaScript:** Lógica de manipulação do DOM, validação de formulário e eventos de clique (Menu e Tema).
